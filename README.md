For cool playing you must enter this command in terminal. It's allows output colored characters to terminal.
Command:

reg add HKEY_CURRENT_USER\Console /v VirtualTerminalLevel /t REG_DWORD /d 1 /f

You can read sea battle's rules in repository's description and at beginning of the game.
To start the game you must open "out" folder. After open "artifacts" folder and open naval_battle.exe.
